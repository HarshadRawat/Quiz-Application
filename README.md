# Quiz Application (Java)

## Overview
This is a console-based Quiz Application developed using Java and Object-Oriented Programming.
The application displays multiple-choice questions, takes user input, and calculates the final score.

## How It Works
The program starts from Main.java, which calls the quiz service.
QuestionService displays each question with four options, accepts the user’s choice, and compares it with the correct answer.
After all questions are answered, the final score is printed.

## Project Structure
-Main.java – Application entry point
-Question.java – Defines a quiz question
-QuestionService.java – Handles quiz flow and scoring

## Technologies Used
-Java
-Object-Oriented Programming
-Arrays, Loops, and Scanner

## How to Run
-Compile the program
  javac Main.java
-Run the application
  java Main

## Output
The program displays each question, takes user input, and prints the total score at the end.
